= clox

A bytecode virtual machine for the Lox programming language, written in C.

Based on https://craftinginterpreters.com[Crafting Interpreters].

== Build

[source,bash]
----
make
----

== Usage

REPL mode:
[source,bash]
----
./clox
----

Run a script:
[source,bash]
----
./clox script.lox
----

== Architecture

* Single-pass compiler â†’ bytecode
* Stack-based VM with 256-element stack
* Hash tables for globals and string interning
* Supports: numbers, booleans, strings, variables, operators, print statements
